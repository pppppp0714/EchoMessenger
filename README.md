# (C# 코딩) 에코 메신저
## 개요
- C# 프로그래밍 학습
- 핵심기능: 전송 버튼을 누르면 TextBox에 입력된 내용을 ListBox에 저장하고 TextBox의 내용을 지움, 전송이 끝나면 다시 TextBox에 Focus, Enter 키를 눌러 전송을 수행할 수 있음, 메시지를 입력한 시간을 알 수 있음, 현재 메시지의 수를 알 수 있음, 입력된 메시지를 선택해서 삭제하거나 메시지 내용 전체를 삭제할 수 있음, 메시지를 보낼 때 입력할 수 있는 제한(50자)가 있음 
- 화면구성: 중앙에 입력한 텍스트를 저장하는 ListBox, 입력을 받는 TextBox, 전송을 담당하는 Button, Echo Messenger 제목을 표시하는 Label, 현재 메시지의 수를 알려주는 Label, 선택한 메시지를 지울 수 있는 Button, 메시지의 전체 내용을 초기화할 수 있는 Button
## 실행 화면
1단계 코드의 실행 스크린샷
![과제1 실행화면](img/screenshot-1.png)
과제 내용
 - Label(제목), ListBox(결과), TextBox(입력), Button(전송) 추가
 - 전송 버튼 클릭 시 TextBox의 입력한 내용을 ListBox의 Items로 추가
 - 추가 직후 TextBox의 내용을 Clear
구현 내용과 기능 설명
 - TextBox에 내용을 입력하고 전송 Button을 누르면 ListBox에 입력한 내용을 저장
 - 전송 후 TextBox의 내용을 비워 추가 입력을 대기
 2단계 코드의 실행 스크린샷
![과제2 실행화면](img/screenshot-2.png)
과제 내용
 - 전송을 Enter 키로 수행
 - 전송 이후 다시 TextBox에 Focus
 - 전송 내용이 공백인 경우 전송하지 않음
구현 내용과 기능 설명
 - Enter 키를 누르면 전송 버튼을 눌렀을 때의 코드를 실행
 - 전송 코드에 TextBox에 Focus하도록 변경
 - 전송 내용이 아무것도 없는 공백일 경우 해당 내용을 전송하지 않음
3단계 코드의 실행 스크린샷
![과제3 실행화면](img/screenshot-3.png)
과제 내용
 - 메시지 앞에 현재 시간을 추가
 - 현재 메시지의 수를 볼 수 있는 텍스트 추가
 - 메시지의 앞 뒤의 불필요한 공백을 없애는 기능 추가
구현 내용과 기능 설명
 - 메시지를 입력하면 메시지 앞에 [시간:분:초] 형식으로 현재 시각을 추가해 입력됨
 - 새로운 라벨을 하나 생성하고 그 라벨의 현재 리스트박스의 아이템들의 수를 반영하여 입력함
 - 입력한 메시지에서 앞 뒤의 불필요한 공백이 있으면 이를 제거한 문자열을 저장해 그 문자열을 결과로 사용함
4단계 코드의 실행 스크린샷
![과제4 실행화면](img/screenshot-4.png)
과제 내용
 - 입력된 메시지를 선택해 삭제하는 기능 추가
 - 전체 메시지를 한번에 삭제하는 기능 추가
 - 메시지 입력 시 최대 글자 제한 추가
구현 내용과 기능 설명
 - ListBox에서 입력된 메시지를 선택할 수 있도록 하고 삭제 버튼을 눌러 그 내용을 지울 수 있도록 함
 - 채팅 내용을 전부 삭제하여 초기화하는 버튼을 추가, 클릭 시 확인 메시지 출력
 - 메시지 입력 시 글자 제한(50자)를 추가하고 이를 넘으면 최대 글자 제한에 도달했음을 알리는 메시지 출력
